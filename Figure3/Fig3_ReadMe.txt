Read me file for Figure 3 


Data related to Figure 3 is on the repository on the folder Data_Fig3_PPC

To reproduce the graphs in figure 3 , run the following mfiles:

Fig3AF_Prototype_fig3_Connectivity_PPC_ok_v1  (Plots the medial & lateral motor areas PPC + Granger) 

Fig3GL_Proto_FigSupple4_Connectivity_PPCS1_ok (Plots the S1 & motor areas PPC +Granger) 


The script that calculates the PPC and Granger is the following: 

TwoMonkeys_stats_mod_TvsLS_copy

For it to run it you need the Data per sessions contained in the following folders:

Data_PPC_Fig3_PerSession_1to5_LS  
Data_PPC_Fig3_PerSession_6to10_LS 
Data_PPC_Fig3_PerSession_11to15_LS 

Data_PPC_Fig3_PerSession_1to5_T 
Data_PPC_Fig3_PerSession_6to10_T 
Data_PPC_Fig3_PerSession_11to15_T
