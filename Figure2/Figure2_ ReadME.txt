Read me file for Figure2 code


Data related to Figure 2 is on the repository in the folder Data_Figure2.

To reproduce the graphs in figure 2, run the mfiles in the following order:

1) For plotting all panels of Figure 2B run:

Plot_perAreasICS_allCleanedLFP_Art (mean eLFP (zScore vs Time)

2) For obtaining the delta between Peaks - Through using a fixed onset of 6 ms run:

Get_dPT_AUC_InCleanwM3Rec_LFPs_Fx6msTime_Rebuttal2 

3) For plotting all DeltaPk-Tr values run: 
Plot_ConfusionMatrix_FixedOnset6ms 

4) For making all the final figures run:
Makefigures_rebuttal2 


* Note: the color code in the figures will be different from the one of the Manuscript figure but it doesn't matter since the colors are consistent across areas.
